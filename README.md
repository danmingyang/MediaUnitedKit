# MediaUnitedKit

**MediaUnitedKit 集成：**

1、图像、视频采集<支持对焦、双击缩放镜头、前后置摄像头切换、闪光灯设置以及屏幕旋转>；

2、自定义相册的创建及视频、图片保存到自定义相册；

3、图片编辑<支持不规则裁剪、旋转、加框、黑白、撤销>；

4、图片选择器；

5、音频的录制、播放<支持本地和网络音频播放>。
